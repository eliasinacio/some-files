#### curl 
É usado para testar conexões e simular as requisições de um client

response body:
`curl <url>`

verbose response (explicação detalhada):
`curl -v <url>`

response completa (header e body):
`curl -i <url>`

request usando um verbo (adiciona -i para receber o header):
`curl -X <VERBO> <url>`

response header:
`curl -I <url>`
